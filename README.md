# RAT - Hodin_3.0 - Alpha

A Remote Administration Tool for Linux

# Dependencies : 
---------------
Install libgtk2.0-dev from your default packet manager.

Install file corresponding to your distrib from folder : Dependencies_installer (clt_Hodin_3.0 folder)
